# 🎬 Netflix Data Analysis

A data analysis project exploring Netflix movies and TV shows using Python and data visualization techniques.

## 📌 Project Overview

This project analyzes the Netflix dataset to discover trends in content types, release years, ratings, genres, and countries. The analysis includes data cleaning, exploratory data analysis, and visualizations.

## 🛠️ Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

 
 📊 Key Insights

* Movies are more common than TV Shows on Netflix.
* The United States contributes the most content.
* Drama and International genres are among the most popular.
* Netflix content has grown significantly over recent years.

## 📁 Project Files

* `netflix_titles.csv` — Dataset
* `Netflix_Data_Analysis.ipynb` — Analysis notebook

## 🚀 How to Run

```bash
git clone https://github.com/yourusername/Netflix-Data-Analysis.git
```
